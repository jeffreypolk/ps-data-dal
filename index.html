ps-data-dal

Library for data access

View project on GitHub

Overview

This project is a library for data access. It includes a single class to access data from SQL Server. This could be enhanced overtime to provide access to other data sources - Excel, Oracle, etc. It was created as a helper to eliminate all the redundant code required for data access - creating connections, building command objects, adding parameters, etc.

Initialization

The constructor accepts a connection string. If none is passed, it will try to pull this from the cnnstr appSetting in the application config file.

Execute method

The Execute method has numerous overloads and returns different types of data based on the output parameter.

Output Writer

This is a collection of classes to redirect output to a place of your choosing. This is handy when you have a background worker thread and you want to send output to the UI. It is a violation to access the UI thread from a background thread. Therefore, something as simple as updating a textbox from a background thread requires delegates, etc. This complexity is abstracted away through these classes.

ProcessRunner

This class makes it easy to initiate external processes. It wraps the System.Diagnostics.Process class. The StandardOutput and StandardError streams are redirected to single location for easy access. It also accepts an Output.IOutputWriter to present the output to the UI thread.
