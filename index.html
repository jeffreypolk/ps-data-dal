ps-data-dal

Library for data access

View project on GitHub

Overview

This library for simple data access. It includes a single class to access data from SQL Server. It was created as a helper to eliminate all the redundant code required for data access - creating connections, building command objects, adding parameters, handling transactions etc.

Initialization

The constructor accepts a connection string. If none is passed, it will try to pull this from the cnnstr appSetting in the application config file.

Execute method

The Execute method has numerous overloads and returns different types of data based on the passed return parameter.

Parameters

Parameters are set using the Parameters.AddWithValue method.

Transaction Handling

A transaction is managed by calling the BeginTransaction method. All subsequent Execute calls are included as part of that transaction. End the transaction by calling the CommitTransaction method or the RollbackTransaction method.
